参考にしたサイトジョジョ卓手引き

スタンド使いにしたときの補正
プロフィールの「スタンド使い？」のチェックをYES!にすることで得られるプラス補正がこちら

  • POWの値が2倍(MP、SAN、幸運の値は1倍のまま)
  • HPの値が2倍
  • 投擲、登攀、跳躍の初期値が50になる
  • HPの自然回復が1週間に一度ではなく1日に一度となる

スタンド使いのステ振り
主に特殊なステ振りはスタンドのパラメータのみ。
パラメータによるスタンドの性能補正は例として以下を示すが、ハウスルール等でいくらでもやりやすいように改変していただいて構わない。

  • POWの値をスタンドPとし成長性以外の5つのパラメータに振り分ける(最大消費Pは25)
  • パラメータによって補正がもらえる
    • 破壊力
      • 高ければ高いほどスタンドのダメージボーナスと防御可能量が上がる
      防御可能量とは、攻撃に対し攻撃で返した際にカットできるダメージ量であり、書かれたダイスを振って出た目の数だけダメージを軽減することができる。
    • スピード
      • 読んで字のごとくスタンドの素早さなので、スタンド自体のDEXと回避技能にあてはめられる。
      DEXは最小が4、パラメータが1つ上がるごとに4づつプラスされていくため最大が20(変更する可能性あり)
      回避技能は最小で0、パラメータが1つ上がるごとに15づつプラスされていくため最大が60(変更する可能性あり)
      ラッシュ攻撃のこぶしの数にも影響する
    • 持続力
      • ここではスタンドのビジョンを出すのにかかるコスト量として考える。
      コストはMPで換算し持続力が高ければ高いほど消費MPが少なくなる。
      最小は0、最大は4
    • 射程距離
      • 今のところ特にステータスに補正は無いが、本体からどのくらいの距離を離れられるか。
      ただ具体的な距離はサイトによってまちまちだったためハウスルールで決めるといいかもしれない。
    • 精密動作性
      • スタンドがどれくらい精密な動作を行えるのか。
      細かい補正をどんどんつけていくとキリがないので、ここではスタンドの持つ技能を戦闘系と動作系に分けてそれぞれに補正値をプラスすることにした。
      戦闘は最小-20、最大+20。動作系(行動系)は最小-30、最大+30とする。
    • 成長性
      • 特に補正も振り分けポイント消費もなし。
      設定を詰め込んだり成長していくスタンドにする場合は高めに設定しておくとフレーバーが濃くなる。
  • スタンドの技能は必要だと思ったものは好きに増やして構わないが、職業Pや興味Pで上乗せすることは出来ない。
  • ただし、クリティカルやスペシャル判定での成長ロールは振ることができる。あとはハウスルールで。

ラッシュ
ラッシュ攻撃をする場合、まずスタンドのタイプが近距離パワー型であることが必要になる。
一方的にラッシュする場合はスタンドのDEX発のパンチ、もしくはキックになるため、その数だけこぶしかキックのダイスロールを行う
そして[成功した数-ファンブルの数]だけダメージロールを行う。
ただし相手に装甲があった場合は当たった数×装甲の分だけダメージ軽減が行われる。

例:スタープラチナ　破壊力A・スピードA → ダメージボーナス3d6・DEX20　こぶし70
敵の装甲 → 5
こぶし判定を20回 → (68,10,2,25,64,95,4,17,66,77,93,44,47,80,43,99,81,50,59,17) → 成功数14 - ファンブル数1
ダメージ → (1*13)d3 + (3*13)d6
→ 28[1,2,3,3,2,1,2,2,3,3,2,3,1]
+144[2,1,2,3,6,1,3,1,4,1,3,3,5,5,6,4,5,1,3,1,5,5,2,2,5,3,6,6,5,4,4,4,6,6,3,4,4,1,1,5,3] = 172
敵の装甲*14 → 70
最終ダメージ → 172 - 70 → 102

ラッシュに対し対抗ラッシュがある場合は先にお互い上記の計算を行いダメージを算出する。
その後お互いのダメージ量で対抗ロールを行う。
6版の対抗計算表は
50 + 能動側能力値*5 - 受動側能力値*5 = 能動側対抗成功率
これをダメージ対抗に置き換えると
50 + 能動側ダメージ量*5 - 受動側ダメージ量*5 = 能動側対抗成功率
となる。
しかし先ほどのラッシュのダメージ計算では数ポイントのズレで自動成功や自動失敗になってしまうので*5は外して計算する。
最終的な計算式はこちら
50 + 能動側ダメージ量 - 受動側ダメージ量 = 能動側対抗成功率

例:スタープラチナ VS ザ・ワールド　スタープラチナ側ダメージ → 172　ザ・ワールド側ダメージ → 141
50 + 172 - 141 = 50 + 31 = 対抗成功率81

この対抗ロールに成功した場合、成功側のダメージ量の半分を相手に与えることができる。
相手に装甲がある場合は前述と同じように当たった数×装甲の分÷2だけダメージから引かれる。
対抗ロールに失敗した場合は相手が上記と同じ処理でダメージを与えてくる。

吸血鬼について
クトゥルフにデフォルトでいる吸血鬼や屍生人はジョジョ卓だと全然雑魚なので独自の補正を行う。

  • CONとSTRをプラス10
  • 投擲、登攀、跳躍、キックの初期値が50になる
  • 攻撃を受けた時即座に2ポイント回復
  • こぶしを当てることで吸血可能
  • 現在のHPが最大値より低い場合1時間に1ポイント回復する

波紋について
波紋は一応武道・仙道の部類に入るが、これはジョジョ卓専用キャラシ作成所なので波紋という一つの技能として換算することにする。
原作の波紋は治療に使ったりもできるためその事もここに記載しておく。
この場合例として私独自のポイントの振り方を記載しておくことにする。

正攻法で取得した際の成功率初期値は
(CON+SIZ)*波紋を修行した期間(月)/15 (小数点切り上げ)
例:CONが10、SIZが14で修業期間が36ヶ月(3年)の場合
(10+14)*36/15 = 24*36/15
= 864/15
= 57.6
= 57 → 波紋の成功率初期値は57

特定の師匠にスパルタ指導で修行してもらい取得した場合の成功率初期値は
(CON+SIZ)*波紋を修行した期間(日)/5 (小数点切り上げ)
例:CONが10、SIZが14で修業期間が14日(2週間)の場合
(10+14)*14/5 = 24*14/5
= 336/5
= 67.2
= 67 → 波紋の成功率初期値は67

その他プレイ中に波紋技能成長に当たるイベント

  • 一度の戦闘で波紋を10回使用する(+1d3)
  • クリティカル(+1d10+5)
  • スペシャル(+1d6+3)

波紋を使った技の判定
波紋は武道に当たるので成功すればこぶしやキックにダメージボーナスをつけることができる。
波紋の判定に成功すればダイス一つ追加、3ポイントの波紋ダメージを加えることができる。(2d3+3+db)
失敗した場合は呼吸が乱れていたということでダイス追加は無しだが1ポイントの波紋ダメージを加えることができる。(1d3+1+db)

相手が吸血鬼や屍生人の場合は成功したときの波紋ダメージが1d10+2になる。
合計ダメージ量が10を超えたら波紋疾走判定でRPで叫んでもいいかもしれない。

治療に使う場合は治療で波紋を使うことを宣言してからダイスロールしてもらう。
成功すれば2d3の回復を行うことができる。
HP回復はできるが、SAN値回復をすることはできない。

波紋戦士のSAN値事情
波紋は恐怖に打ち勝つというので、波紋技能50以上でダイスロール権を得て-10補正後の波紋成功率で成功すればSANチェック軽減もしくは無効になる。
もちろんこの辺りはハウスルールで好きに決めてもらっていいし、SANチェック介入は不可能ということにしても構わない。

その他波紋を使った技について
作中で使われたズームパンチや仙道疾走連打等の技を取り入れたい場合はKP(GM)に相談しながら独自の判定方法を考えるといいだろう。
